# Termius: SSH-клиент для тех, кто устал от консоли

## Что это такое

Termius — это кроссплатформенный SSH-клиент с графическим интерфейсом, облачной синхронизацией и поддержкой всех основных операционных систем: Windows, macOS, Linux, Android и iOS. В отличие от стандартного `ssh` из терминала, Termius предлагает удобное управление хостами, сниппетами и ключами.

## Основные возможности

| Функция | Описание | Пример использования |
|---------|----------|----------------------|
| Облачная синхронизация | Хранение сессий и ключей в облаке | Переключился с ноутбука на телефон — все хосты уже там |
| Snippets | Шаблоны команд с переменными | `deploy --env={{environment}} --tag={{version}}` |
| Port forwarding | Проброс портов через GUI | Проброс `3306` с удалённого сервера на `localhost:33060` |
| SFTP клиент | Встроенный файловый менеджер | Копирование логов без `scp` и команд |
| Key agent | Хранение `ssh` ключей с паролем | Ключи никогда не покидают зашифрованное хранилище |
| TOTP | Встроенный генератор одноразовых кодов | Для серверов с двухфакторной аутентификацией |

## Интересные сценарии использования

*   **Управление Raspberry Pi с телефона** — Termius на Android позволяет подключиться к Pi по SSH даже без внешнего экрана. Можно перезагрузить, посмотреть логи, запустить `docker compose up -d`. Добавьте сниппет `temp` с командой `vcgencmd measure_temp` — и температура процессора всегда под рукой.

*   **Резервный доступ к домашнему серверу** — настройте на роутере проброс порта (нестандартный порт, например `54321`). В Termius создайте хост с публичным IP, укажите ключ без парольной фразы. Если основной ноутбук сдох, открываете телефон и останавливаете зависший `ffmpeg`.

*   **Автоматизация деплоя через Snippets** — сниппет `deploy` с переменными:

```bash
cd /var/www/{{project}}
git pull origin {{branch}}
npm install
pm2 restart {{project}}
```

Переменные заполняются через GUI перед отправкой. Ошибки подсвечиваются красным прямо в терминале.

*   **Мониторинг через порты** — в Pro-версии можно открыть несколько вкладок одновременно: в одной `htop`, в другой `tail -f /var/log/nginx/access.log`, в третьей `journalctl -fu docker`.

## Где хранятся данные (важно для безопасности)

| Тип данных | Free (локально) | Pro (облако) |
|------------|----------------|--------------|
| Список хостов | Зашифрован на устройстве | Синхронизируется через сервера Termius |
| SSH ключи | В системной связке ключей | Только в зашифрованном виде, расшифровка на устройстве |
| Snippets | ❌ | На серверах Termius (шифрование на клиенте) |
| История команд | В локальной базе SQLite | Не синхронизируется |

> Ключи Termius не видит никогда. Даже при облачной синхронизации ключи шифруются вашим мастер-паролем на устройстве, и только зашифрованный блоб улетает в облако.

## Недостатки и ограничения

*   **Бесплатная версия** — нет сниппетов, нет SFTP вкладки, нет порт-форвардинга через GUI. Это сильно урезает удобство.
*   **Закрытый исходный код** — нельзя проверить, как именно реализовано шифрование и синхронизация.
*   **Нет плагинов** — нельзя расширить функциональность своими скриптами (в отличие от `Tabby` или `WindTerm`).
*   **Подписка** — Pro версия стоит денег, и это не разовая покупка.
*   **Автообновления** — Termius обновляется автоматически, и иногда после обновления ломаются сниппеты из-за изменения формата хранения.

## Советы для эффективной работы

1. **Группируйте хосты по тегам** — `prod-db`, `prod-web`, `staging`. В большом списке легче ориентироваться.

2. **Настройте Key agent сразу** — создайте пару ключей внутри Termius (Tools → Key Generator) и добавьте публичный ключ на сервер. Парольную фразу запомнит Termius, вводить её каждый раз не придётся.

3. **Используйте сниппеты для длинных команд** — вместо `docker logs --tail=100 --follow $(docker ps -qf "name=app")` сохраните сниппет `logs` и вызывайте через Ctrl+Shift+P.

4. **Включайте подтверждение перед закрытием вкладки** — в настройках есть опция `Confirm before closing`. Спасёт, если случайно нажали Ctrl+W с незаконченным `apt upgrade`.

5. **На телефоне включите внешнюю клавиатуру** — если используете iPad или Android с клавиатурой, в настройках Termius есть раскладка с Ctrl, Alt, Esc и стрелками. Обычная клавиатура без Fn не даст нажать, например, `Ctrl+C`.

## Заключение

Termius — это не просто SSH-клиент. Это менеджер инфраструктуры для тех, кто работает с серверами не из-под Linux каждый день. Студенту на Windows, который подключается к универскому серверу, разработчику с iPad в командировке, команде из трёх человек без общего VPN — каждому найдётся сценарий. Бесплатная версия полезна, но настоящая мощь открывается с Pro: сниппеты, вкладка SFTP и порт-форвардинг в несколько кликов. Если вы устали запоминать IP-адреса и пути к ключам, Termius решит эти проблемы. Если вам нужен бесплатный и открытый клиент — присмотритесь к Tabby. Но удобство синхронизации между телефоном и компьютером стоит тех денег, которые просит Termius.
