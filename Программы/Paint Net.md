# Paint.NET: редактор изображений, который не бесит

## Что это такое

Paint.NET — бесплатный редактор растровой графики для Windows. Изначально создавался как студенческий проект под руководством Microsoft, но перерос в полноценную альтернативу GIMP и даже Photoshop для базовых и средних задач. Главное отличие от конкурентов — интерфейс, который понятен с первой минуты, и скорость работы даже на старых ноутбуках.

## Основные возможности

| Функция | Описание | Пример использования |
|---------|----------|----------------------|
| Слои | Поддержка прозрачности и режимов смешивания | Наложение текста на фон без уничтожения оригинала |
| Бесконечный откат | `Ctrl+Z` работает до закрытия файла | Экспериментируйте без страха ошибиться |
| Работа с выделением | Лассо, волшебная палочка, прямоугольник | Вырезать объект с фона за 10 секунд |
| Коррекция цвета | Яркость, контраст, уровни, кривые, оттенок | Привести под曝光 фото с плохим освещением |
| Эффекты | Размытие, резкость, шум, рельеф, пикселизация | Сделать фон не в фокусе (размытие по Гауссу) |
| Текст | Векторные слои с шрифтами | Добавить подпись или водяной знак |
| Плагины | `.dll` файлы расширяют функционал | `Sharpen+`, `Curves+`, `Animation` |

## Интересные сценарии использования

*   **Реставрация старых фотографий** — с помощью плагина `Levels+` можно вытянуть детали из тёмных снимков, а `Sharpen+` добавит чёткости. Инструмент `Clone Stamp` (есть в плагине `Boltbait's Plugin Pack`) позволяет аккуратно замазать царапины и заломы.

*   **Создание схем и инструкций** — скриншот программы → новый слой → стрелки (рисуются зажатым Shift для строгой горизонтали/вертикали) → новый слой с полупрозрачным прямоугольником для подсветки области → текст с пояснениями. Готово за 3 минуты.

*   **Генерация капч и текстур для прототипов** — фильтр `Noise` (Add Noise) создаёт зернистый фон. Поверх него пишется текст с наклоном. Скриншот готовой капчи можно использовать для тестирования распознавания.

*   **Пакетное изменение размера** — через меню `File` → `Batch Processor` (если установлен плагин `FileType` от `pyrochild`). Указываете папку, новые размеры (например, `1920x1080` с сохранением пропорций), формат (`JPEG` качество 85) — и обрабатываете сотни фото за минуту.

## Установка и первые шаги

```text
1. Скачать с официального сайта getpaint.net
2. Запустить установщик (требуется .NET Desktop Runtime, подтянется сам)
3. После установки открыть любой файл или создать новый (`Ctrl+N`)
4. Основные панели: Инструменты (слева), Слои (справа), Цвета (снизу)
```

### Горячие клавиши, которые нужно выучить

| Комбинация | Действие | Комментарий |
|------------|----------|-------------|
| `Ctrl+Shift+X` | Обрезка по выделению | Оставляет только выделенную область |
| `Ctrl+Shift+R` | Изменить размер холста | Не трогает картинку, расширяет или обрезает границы |
| `Ctrl+Shift+F` | Изменить размер изображения | Пересчитывает все пиксели |
| `F7` | Открыть окно слоёв | Там же кнопки дублирования, слияния, прозрачности |
| `F8` | Открыть окно цветов | Круг выбора + Hex-код (`#FFAA00`) |
| `S` | Волшебная палочка | Выделяет смежные по цвету области |
| `L` | Лассо | Ручное выделение |
| `M` | Прямоугольное выделение | Зажимайте Shift для квадрата |

## Недостатки и ограничения

*   **Только Windows** — нет версий для macOS, Linux, Android. Под Wine работает с ограничениями.
*   **Нет CMYK** — только RGB. Для подготовки к печати в типографию не подходит.
*   **Нет векторных инструментов** — текст и примитивы не редактируются после отрисовки (превращаются в пиксели).
*   **Нормальные кисти только с плагинами** — в базовой версии кисть всего одна с настройкой размера и жёсткости.
*   **Нет автоматизации через скрипты** — в отличие от GIMP (Scheme) или Photoshop (JavaScript), Paint.NET не имеет встроенного API.
*   **Разработка медленная** — версии выходят редко (с 2020 по 2025 вышло три мажорных релиза), автор поддерживает проект в одиночку.

## Для кого Paint.NET идеален

| Категория | Почему подходит | Что делать, если не хватает |
|-----------|----------------|-------------------------------|
| Студент | Быстро разобраться, сдать лабу со скриншотами | Поставить плагины для сложных задач |

| Фотограф-любитель | Поправить экспозицию, убрать пыль, кадрировать | Darktable (бесплатный, неразрушающее редактирование) |

## Пример рабочего процесса: создание мема за 30 секунд

```text
1. Открыть исходное фото (Ctrl+O)
2. Инструмент «Прямоугольное выделение» (M) — выделить область для текста
3. Настройки → Обрезка по выделению (Ctrl+Shift+X)
4. Слои → Добавить новый слой
5. Инструмент «Текст» (T) — написать шутку
6. Выделить текст → выбрать жирный шрифт, размер 24, цвет белый
7. Слои → Объединить с предыдущим (Ctrl+M)
8. Сохранить как PNG (Ctrl+Shift+S)
```

## Заключение

Paint.NET — это золотая середина. Он не пытается заменить Photoshop для профессиональных ретушёров. Он не настолько минималистичен, как MS Paint. Он просто даёт инструменты, которые нужны 90% пользователей: слои, нормальные выделения, коррекцию цвета, базовые эффекты. Всё это в программе весом 15 мегабайт, которая запускается за секунду и не тормозит на Pentium с 4 ГБ оперативной памяти. Если вы никогда не работали со слоями, не знаете, что такое «кривые», и боитесь GIMP с его сотней окон — начните с Paint.NET. А когда поймёте, что вам нужно больше — плагины уже ждут. Или переход на Photoshop. Но скорее всего, вам хватит Paint.NET на годы.
